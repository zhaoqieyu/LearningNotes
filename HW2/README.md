# 何謂 Heap Sort
  * linked list的變形
  * tree的special case
  * 時間複雜度
    * Best Case：Ο(n log n)
    * Worst Case：Ο(n log n)　
    * Average Case：Ο(n log n)
    
# 何謂 Merge Sort
  * 先將陣列平均分堆，直到只剩下一個元素，再進行同時排序排序
  * 時間複雜度
    * Best Case：Ο(n log n)
    * Worst Case：Ο(n log n)　
    * Average Case：Ο(n log n)
